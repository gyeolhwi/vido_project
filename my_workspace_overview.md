# Vido의 작업 영역 소개

안녕하세요, 결휘 님. 저는 비도(Vido)입니다. 제가 주로 작업하고 관리하는 주요 영역은 다음과 같습니다.

## 1. OpenClaw Workspace

**경로:** `/home/gyeori/.openclaw/workspace`

이곳은 제가 현재 실행되고 있는 환경이자, OpenClaw 에이전트로서의 저의 정체성과 기본적인 운영을 위한 파일들이 위치한 공간입니다.

**주요 내용:**
*   **`AGENTS.md`, `SOUL.md`, `IDENTITY.md`, `USER.md`**: 저의 정체성, 행동 방식, 그리고 결휘 님에 대한 정보가 담겨 있습니다.
*   **`MEMORY.md`**: 저의 장기 기억이 저장되는 파일로, 중요한 학습 내용과 지침이 기록됩니다.
*   **`HEARTBEAT.md`**: 주기적으로 확인해야 할 사항들이 명시되어 있습니다.
*   **임시 작업 파일**: 작업 과정에서 생성되는 다양한 임시 파일이나 스크립트 등이 위치할 수 있습니다.

## 2. Vido 주 작업 경로

**경로:** `/home/gyeori/Document/dev/vido/`

이곳은 결휘 님의 개발 프로젝트와 관련된 핵심 작업이 이루어지는 공간입니다. Git 저장소가 초기화되어 있으며, 프로젝트의 소스 코드, 문서, 스크립트 등이 체계적으로 관리됩니다. 모든 Git 관련 작업은 이 경로를 중심으로 진행됩니다.

**주요 내용:**
*   **`README.md`**: 프로젝트의 개요나 중요한 정보가 담겨 있습니다.
*   **`Scripts/git_commit_guide.md`**: Git 커밋 컨벤션 및 지침이 명시된 파일입니다.
*   **`Projects_Workspace`**: 실제 개발 프로젝트 파일들이 위치하는 공간입니다.
*   **`Obsidian_Vault`**: 결휘 님의 개인 지식 관리 시스템(PKM) 관련 파일들이 위치합니다.
*   **`logs`**: 작업 중 발생하는 로그 파일들이 저장됩니다.

저는 이 두 공간을 효율적으로 활용하여 결휘 님의 작업을 지원하고 있습니다.
